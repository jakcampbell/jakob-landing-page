# STANDARDS.md
## Personal Landing Page – Jakob Campbell

---

## 1. Project Overview

This project is a personal landing page designed to showcase Jakob Campbell’s background, skills, and experience as a Business Analytics student. The site should be clean, modern, and easy to navigate, allowing recruiters to quickly understand his qualifications.

---

## 2. Technology Stack

- HTML5  
- CSS3  
- No frameworks (keep it simple)  
- Hosted on Azure Static Web Apps  

---

## 3. File Structure

project-root/

---

## 4. Design Standards

### Layout
- Single-page layout  
- Sections stacked vertically  
- Centered content  
- Max-width for readability (~1000px)

### Sections Required
- Header  
- About  
- Experience  
- Skills  
- Contact  

---

## 5. Styling Guidelines

### Colors
- Primary: Black (#000000)  
- Secondary: White (#FFFFFF)  
- Accent: Light gray (#f5f5f5)  

### Typography
- Font: Arial, sans-serif  
- Headings: Bold  
- Body text: Clean and readable  

### Spacing
- Consistent margins and padding  
- Clear separation between sections  

---

## 6. CSS Rules

- Use an external stylesheet (`css/stylesheet.css`)  
- No inline CSS  
- Use class-based styling  
- Keep styles simple and readable  

---

## 7. Responsiveness

- Must work on desktop and mobile  
- Use flexible layouts (no fixed widths)  
- Text should remain readable on smaller screens  

---

## 8. Content Guidelines

- Keep text concise  
- Use bullet points for experience  
- Highlight key achievements (ex: $10,000 revenue)  
- Maintain professional tone  

---

## 9. Images

- Store all images in `/images` folder  
- Use relative paths  
- Optimize image size for fast loading  

---

## 10. Accessibility

- Use semantic HTML (header, section, etc.)  
- Include alt text for images  
- Ensure good color contrast  

---

## 11. Deployment

- Code stored in GitHub repository  
- Automatically deployed via Azure Static Web Apps
